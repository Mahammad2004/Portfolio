# Mahammad Basha - Personal Portfolio

A modern, responsive, and interactive personal portfolio website built to showcase my skills, experience, and projects as a Full Stack Developer.

This project is built using a modern tech stack focused on performance and developer experience:

-   **[React](https://react.dev/)**: JavaScript library for building user interfaces.

## âœ¨ Features

-   **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop screens.
-   **Smooth Navigation**: Sticky navbar with smooth scrolling to sections and active state highlighting.
-   **Interactive Animations**: Engaging scroll-triggered animations using Framer Motion.
-   **Glassmorphism UI**: Modern aesthetic with glass-like effects and vibrant gradients.
-   **Sections**:
    -   **Hero**: Dynamic introduction with a typing text effect.
    -   **About**: Profile snapshot highlighting education and key stats.
    -   **Skills**: categorized view of technical expertise (Frontend, Backend, Tools).
    -   **Experience**: Chronological timeline of internships and work history.
    -   **Projects**: Showcase of key projects with problem/solution breakdowns.
    -   **Achievements**: Highlights of hackathon wins and academic rankings.
    -   **Contact**: Multiple channels to get in touch (Email, Phone, Linear/GitHub).

## ğŸ› ï¸ Installation & Setup

Follow these steps to run the project locally on your machine.

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Mahammad2004/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.

4.  **Build for Production**:
    ```bash
    npm run build
    ```
